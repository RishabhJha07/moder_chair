## ✨ Features

- 🎨 Chair color selection using radio buttons (pure CSS)
- 🧾 Toggle between **Description** and **Details**
- 🛒 Styled “Add to Cart” UI
- 💫 Smooth transitions and animations
- 📱 Fully responsive (media queries included)
